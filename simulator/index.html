Command Reference
Command Syntax Description
inin Re-initialise the file system
crcr <name> Create a file (alphanumeric name)
dede <name> Destroy a file — must be closed first
opop <name> Open a file → returns OFT slot index
clcl <slot#>Close file at OFT slot (number, not name)
wmwm <pos> <string>Write string into memory buffer M
rmrm <pos> <count>Read count bytes from memory
wrwr <slot#> <mpos> <count>Write from M to open file
rdrd <slot#> <mpos> <count>Read from open file into M
sksk <slot#> <pos>Seek file position pointer
drdr List all files and their sizes
common mistakes:
cl takes a slot number — cl 1 not cl myfile
wr / rd / sk first arg is the OFT slot, not the filename
de will fail if the file is still open — cl first
